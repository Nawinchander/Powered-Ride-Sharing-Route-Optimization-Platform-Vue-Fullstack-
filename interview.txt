Interview Questions
Why hash passwords?

Passwords should never be stored in plaintext because database breaches would expose user credentials.

Why JWT instead of sessions?

JWT scales better in distributed systems because the server does not need to maintain session state.

Why Repository Pattern?

Repositories separate business logic from database logic which improves testability and maintainability.

Interview Questions
Authentication
Why use Access Token and Refresh Token together?

Access tokens should be short-lived for security, while refresh tokens allow seamless re-authentication without forcing users to log in repeatedly.

Why not store JWT in Local Storage?

Local storage is vulnerable to XSS attacks. Secure HTTP-only cookies are generally safer for production applications.

Why RBAC instead of hardcoded permissions?

Role-based access control scales much better as applications grow and more user roles are introduced.
